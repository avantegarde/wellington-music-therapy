Add Gravity Forms data to the "Export Personal Data" and "Erase Personal Data" tools.

== Description ==

When you use the "Export Personal Data" under tools in WP 4.9.6+, Gravity Forms data will be included in the export.
If you "Erase Personal Data" this will remove the entries from the relevant entries table in Gravity Forms.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Make sure you have Gravity Forms installed and active
2. Activate the plugin
3. Export or delete data as needed
